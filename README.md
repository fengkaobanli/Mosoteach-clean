# Mosoteach-clean 辅助工具

## 项目简介

Mosoteach-clean 辅助工具是一个为云班课平台设计的自动化工具，旨在帮助用户更高效地管理和学习平台上的课程资源。该工具支持自动刷课功能，为用户提供便捷的学习体验。

## 功能特点

- **自动登录**：支持使用账号密码自动登录云班课平台
- **资源识别**：智能识别音频、视频和其他类型的资源
- **自动刷课**：支持自动刷音频和视频资源，节省学习时间

## 安装步骤

### 前提条件

- Python 3.6 或更高版本
- pip 包管理工具

### 使用方法

```bash
// 下载项目代码
git clone https://github.com/ubadly/Mosoteach.git
// 进入项目目录
cd Mosoteach
// 安装依赖
pip install requests parsel
// 运行主程序
python main.py
```

## 注意事项

1. **账号安全**：请妥善保管您的账号密码，不要在公共场合或不安全的环境中使用
2. **使用频率**：请勿过度使用自动刷课功能，以免被平台检测到异常行为
